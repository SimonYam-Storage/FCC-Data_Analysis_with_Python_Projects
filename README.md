# Data Analysis with Python Projects

This repository contains Data analysis with Python Projects, python project for data analysis. 

These projects includes
1.Mean-Variance-Standard Deviation Calculator: 

* Build a computation function using NumPy to perform the statistics for 3x3 matrices.

2\. Demographic Data Analyzer:

* Perform the data analysis on census datasets using Pandas to extract the insights and demographic trends.

3\. Medical Data Visualizer:

* Conduct data analysis and correlation mapping from medical examination data using Matplotlib and Seaborn to explore the relationship between cardiac disease and human lifestyle habits.

4\. Page View Time Series Visualizer: 

* Analyze and visualize the daily website traffic from dataset using Pandas, Matplotlib and Seaborn to understand on trend identification and identify the growth

5\. Sea Level Predictor:

* Develop a predictive model using SciPy linear regression to forecast global sea-level changes through 2050 from the historical trends.
